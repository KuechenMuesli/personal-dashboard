# Entitäten

Dieser Bereich dokumentiert die zentralen Datenentitäten des Systems, die im [Datenmodell](../architecture/data-model.md) definiert sind.

*   [User](./user.md) - Die zentrale Benutzerentität.
*   [Layout](./layout.md) - Repräsentiert ein Dashboard-Setup (Workspace).
*   [Widget](./widget.md) - Das Konzept eines Widgets (Definition und Instanz).
*   [UserSecrets](./user-secrets.md) - Der flexible Speicher für Konfigurationen und Geheimnisse.
*   [CustomTheme](./custom-theme.md) - Ein vom Benutzer erstelltes Theme.
*   [Quickshare](./quickshare.md) - Ein temporärer, geteilter Inhalt.
